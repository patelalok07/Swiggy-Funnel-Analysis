# 📊 Swiggy Funnel Analysis – 2019 Case Study

## 📌 Project Overview

This project presents a detailed **Funnel Analysis of Swiggy’s 2019 performance**, using session-level, traffic source, and supporting data to evaluate order trends, traffic fluctuations, and conversion efficiency across the customer journey.

The objective is to identify key growth drivers, bottlenecks in the funnel, and provide data-backed recommendations to improve overall business performance.

---

## 🎯 Business Objectives

- Identify weekly order trends (highs & lows)
- Analyze traffic changes and their contributing channels
- Break down the conversion funnel stages:
  - **L2M** (Listing → Menu)
  - **M2C** (Menu → Cart)
  - **C2P** (Cart → Payment)
  - **P2O** (Payment → Order)
- Validate hypotheses using supporting metrics (discounts, delivery charges, payment success rate)
- Provide actionable growth recommendations

---

## 📂 Dataset Description

The analysis is based on three primary sheets:

### 1️⃣ Session Details
- Daily sessions across funnel stages  
- Order counts  
- Listing activity  

### 2️⃣ Channel Wise Traffic
Traffic segmented by source:
- Paid  
- Organic  
- Direct  
- Referral  

### 3️⃣ Supporting Data
- Average discounts  
- Delivery charges  
- Packaging charges  
- Payment success rate  

---

## 📈 Key Analysis Performed

### 1️⃣ Order Trends Analysis
- Identified days with ±20% change in weekly orders  
- Observed recurring weekend spikes  
- Evaluated weekday dips and possible reasons  

### 2️⃣ Traffic Analysis
- Compared daily sessions with the same weekday of the previous week  
- Analyzed source-wise contribution to traffic shifts  
- Correlated promotional activities with paid traffic spikes  

### 3️⃣ Conversion Funnel Analysis

Calculated conversion rates for:

| Funnel Stage | Formula |
|--------------|----------|
| **L2M** | Menu Visits / Listing Sessions |
| **M2C** | Cart Adds / Menu Visits |
| **C2P** | Payment Initiations / Cart Adds |
| **P2O** | Successful Orders / Payment Initiations |

- Identified major drop-off points  
- Correlated fluctuations with:
  - Discount changes  
  - Delivery charges  
  - Payment success rate  

---

## 🔍 Key Insights

- 📅 Weekend spikes significantly increase total orders  
- 📢 Paid campaigns drive short-term traffic boosts  
- 💰 Higher discounts positively impact P2O conversion  
- 🚚 Increased delivery charges negatively affect C2P conversion  
- 📉 Funnel drop-offs highlight optimization opportunities in mid and bottom funnel stages  

---

## 💡 Recommendations

- Offer targeted weekday discounts to stabilize demand  
- Optimize delivery and packaging charges  
- Improve menu presentation (images, descriptions) to increase L2M conversion  
- Strengthen payment reliability to reduce P2O drop-offs  
- Run strategic paid campaigns aligned with low-demand days
